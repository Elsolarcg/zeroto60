Por qué este modelo se paga solo

Todo lo que se construye en la fase de instalación tiene un único objetivo: que el sistema empiece a generar ventas reales lo antes posible.

Cuando el funnel, la infraestructura, la pauta y la lectura de datos funcionan como un sistema integrado, el crecimiento deja de depender de inversión externa constante.

El objetivo es que El Solar deje de ser un costo fijo y pase a ser un socio directamente alineado a resultados.

Modelo económico · Fase 1

Fee de instalación

Durante los primeros 3 meses, se establece un fee fijo para cubrir el trabajo intensivo de construcción, validación y estabilización del sistema.

  • • Inversión: USD 1.950 / mes
  • • Duración: 3 meses
  • • Comisión: No aplica

Esta fase existe para reducir riesgo, evitar errores críticos y permitir que el sistema llegue a un punto de tracción real.

Modelo económico · Fase 2

Modelo a comisión

Una vez el proyecto alcanza una facturación mínima mensual de USD 10.000, el modelo cambia automáticamente.

  • • Fee fijo mensual: Se elimina
  • • Comisión: 10% sobre ventas mensuales
  • • Incentivo: Crecimiento y escalamiento

A partir de este punto, El Solar solo gana si el proyecto crece. Los incentivos quedan completamente alineados.
